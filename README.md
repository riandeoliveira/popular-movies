## 📚 About

An application to search for and favorite movies in the browser's Local Storage, using the [The Movie Database (TMDB)](https://www.themoviedb.org/?language=en) API.

## 🏗️ Development

Project proposed by the challenger [Giovanna Moeller](https://www.linkedin.com/in/giovannamoeller) during the [7 Days Of Code](https://7daysofcode.io/) event by [Alura](https://www.alura.com.br/).

## 🕹️ Features

- Add/remove movies from favorites.
- Filter movies by searched or favorite ones.
- Search for movies by name.

## 📝 Notes

Although the technologies proposed by the challenge were just JavaScript and DOM, I chose to practice using my current stack.

## 💻 How to Access

The application is available online and can be accessed [here](https://popular-movies-six.vercel.app/).
